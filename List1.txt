25) a) D = pi*R^2/(2*pi*R)^2 = 1/(4*pi)
D para um círculo é sempre máximo, pois D = Compacidade^-1, onde na compacidade o valor é mínimo para um círculo. Como nela o valor é mínimo, seu inverso será um valor máximo.
b) D = L^2/(4L)^2 = L^2/(16*L^2) = 1/16 - Quadrado
D = L*(L+d)/(2*L+2*(L+d))^2 = (L^2+d*L)/(4*L+2*d)^2 = (L^2+d*L)/(16*L^2+2*d*L+4*d^2)
Com L = 1, D = (d+1)/(4*d^2+2*d+16)
Conforme a diferença entre o maior lado e o menor aumenta, o denominador 4*d^2+2*d+16 também aumentará em uma proporção muito maior que o numerador d+1, diminuindo o valor. Na compacidade, esse valor sempre aumenta. Como essa diferença maior lado e o menor não pode ser 0, concluimos que D é sempre menor para um retângulo do que um quadrado.
